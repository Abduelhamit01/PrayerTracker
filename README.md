# PrayerTracker

A simple iOS app to track your daily prayers.

## Features

- Track all 5 daily prayers (Fajr, Dhuhr, Asr, Maghrib, Isha)
- Mark Fardh and Sunnah prayers as completed
- Calendar view with color-coded status (green/yellow/red)
- 30-day statistics
- Qibla
- Weekly overview
- Dark mode support
- Available in German, English, and Turkish

## Privacy

PrayerTracker respects your privacy. All data is stored locally on your device - nothing is collected or sent to any server.

See [Privacy Policy](PRIVACY_POLICY.md) for details.

## Requirements

- iOS 17.0+
- Xcode 15.0+
